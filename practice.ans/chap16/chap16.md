### Q21:

**問題**: 数値を受け取り、偶数なら `0`、奇数なら `1` を終了ステータスとして返す `check_even` 関数を記述しなさい。  
**答え**:

```bash
check_even() {
  if [ $(($1 % 2)) -eq 0 ]; then
    return 0
  else
    return 1
  fi
}
```

    return 0

  fi
